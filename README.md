## 📖 عن المشروع

**المازن للكمبيوتر** هو موقع الشركة الرسمي لإحدى أبرز الشركات اليمنية المتخصصة في مجال تكنولوجيا المعلومات. تأسست الشركة عام 2000 وتمتلك أكثر من عقدين من الخبرة في تقديم الحلول التكنولوجية المتكاملة.

### 🏢 نبذة عن الشركة

- **تأسيس**: 2000
- **الخبرة**: أكثر من 20 عاماً
- **التخصص**: حلول تكنولوجيا المعلومات المتكاملة
- **التغطية**: جميع أنحاء اليمن
- **الفروع**: صنعاء، الحديدة، عدن

### 🎯 الرؤية والرسالة

**الرؤية**: أن نكون الشريك الموثوق في عالم تكنولوجيا المعلومات في اليمن والمنطقة.

**الرسالة**: تقديم حلول تكنولوجية متكاملة ومبتكرة عالية الجودة لعملائنا من الشركات والمؤسسات.

## 🚀 البدء السريع

### 📋 المتطلبات الأساسية

قبل البدء، تأكد من وجود المتطلبات التالية:

- 🌐 **متصفح ويب حديث** (Chrome, Firefox, Safari, Edge)
- 🖥️ **خادم محلي** (XAMPP, WAMP, أو أي خادم آخر)
- 📝 **محرر نصوص** (VS Code, Sublime Text, إلخ)

### ⚡ التثبيت والتشغيل

1. **استنساخ المشروع**
   ```bash
   git clone https://github.com/username/almazen-computer.git
   cd almazen-computer
   ```

2. **تثبيت التبعيات**
   ```bash
   npm install
   ```

3. **بناء ملفات CSS**
   ```bash
   npx tailwindcss -i ./style.css -o ./output.css --watch
   ```

4. **تشغيل المشروع**
   - انقل المجلد إلى مجلد `htdocs` في XAMPP
   - شغل XAMPP وفعل Apache
   - افتح المتصفح واذهب إلى: `http://localhost/almazen-computer`

### 🔧 التطوير

للتطوير والتعديل:

```bash
# مراقبة تغييرات CSS
npm run dev

# بناء الإنتاج
npm run build
```

## 📖 كيفية الاستخدام

### 🧭 التنقل في الموقع

الموقع يحتوي على الأقسام التالية:

1. **الرئيسية**: نظرة عامة على الشركة والخدمات
2. **المنتجات**: عرض المنتجات المتاحة
3. **الخدمات**: الخدمات المقدمة
4. **الموردين**: الشركاء والموردين

6. **معلومات الشركة**: نبذة عن الشركة
7. **المدونة**: المحتوى التعليمي
8. **اتصل بنا**: معلومات الاتصال

### 🎨 تخصيص المظهر

للتخصيص:

1. **الألوان**: عدل متغيرات Tailwind في `tailwind.config.js`
2. **الخطوط**: غير الخطوط في ملف CSS
3. **الصور**: استبدل الصور في مجلد `images/`
